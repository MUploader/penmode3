How Do I install Penmode3?

If you have a working Node.js & NPM setup, just type
npm install penmode3
For other instruction read the Readme

How Do I install Penmode3 Plugins?

Download this penmode3-plugin Repo and start penmode3 with penmode3 --plugins /downloaded_folder

When I can use Penmode3? (Use-Case)

  • You want to run your pentest from your powerfull VPS/Server
  • Your (company) firewall does not allow you to access some ports on the target system
  • You don't know how to use your Raspberry Pi
  • More...
